1. Requirements

  1. Install these first:
    • Node.js (version 16 or newer). Download from nodejs.org
    • Code Editor (recommended: VS Code)
  2. Extract the project files to a folder on your computer (for example: C:/Users/YourName/Documents/volley-vista-landing).

2. How to Install

  1. Open the project folder on your computer.
  2. Open a terminal/command prompt in that folder.
    (Tip: In Windows, right-click in the folder > choose Open in Terminal or Open in Command Prompt.)
  3. Type this command and press Enter: 
    npm install
    This will automatically download all the website's requirements (libraries).

3. Run the Website (Development Mode)

  1. After npm install is finished, run: 
    npm run dev
    The website will start in development mode. You will see an address like http://localhost:5173 in the terminal. Click or copy-paste it into your browser.
  2. The website is now live and ready to edit. Any file changes will update automatically in your browser.

4. Important Folder Structure

  • index.html: The main website page.
  • src/style.css: Controls the website's appearance (colors, fonts, layout).
  • src/main.js: Main JavaScript for animations/interactions.
  • src/assets/: Contains images used on the website.
  • src/assets/: Contains icons or extra assets.

5. How to Edit the Website

  1. Edit Text/Content:
    Open index.html in your code editor. Find and change the text you want (title, description, player names, schedule, etc).
  2. Change Images:
    • Put your new image in the src/assets/ folder.
    • Edit index.html and change the src="..." in the <img> tag to your image file name.
  3. Edit Colors/Styles:
    Open src/style.css. You can change colors, fonts, sizes, etc.
    For color and style references, see Tailwind CSS documentation.
  4. Edit Animations/Interactions:
    Open src/main.js to change animation effects, menus, etc.

6. Build/Export for Hosting

  1. When you are done editing, run: 
    npm run build
    A new folder called dist will appear. This contains the website files ready to upload to your hosting.
  2. Upload the contents of the dist folder to your hosting (for example: cPanel, Netlify, Vercel, etc).

7. FAQ (Frequently Asked Questions)

Q: I have no coding experience. Can I use this website?
A: Yes! Just follow this guide step by step. For simple edits (text/images), you do not need to know coding.
Q: What if the website breaks after I edit it?
A: Check your changes. If needed, restore the original file (always make a backup before editing!).
Q: How do I add a new player or schedule item?
A: Edit index.html, copy-paste a player/schedule block, and change the details as needed.
Q: Can I get help if I am stuck?
A: Of course! Contact the website seller/developer for further assistance.

8. Need Help?

If you have any trouble, please contact the website seller/developer for more guidance.

Tip: Always make a backup before making big changes!
Warning: Do not delete important files like package.json, vite.config.js, tailwind.config.js, or the src/ folder.
